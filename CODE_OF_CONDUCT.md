# Contributor Covenant Code of Conduct

This project follows the Contributor Covenant v2.1. By participating, you agree to abide by its terms.

Key points

- Be respectful and considerate in communications.
- Be collaborative and constructive with feedback.
- Harassment, discrimination, or abusive language will not be tolerated.

For the full text and reporting guidelines, see https://www.contributor-covenant.org/version/2/1/code_of_conduct/
